# 🚀 SubTrackr – Subscription Management System

A full-stack web application to track, manage, and analyze subscriptions in one place.  
It helps users monitor spending, get renewal alerts, and avoid unnecessary charges.

---

## 📌 Features

### 🔐 Authentication
- User registration and login  
- Secure authentication with protected routes  
- Session handling  

### 📊 Dashboard & Analytics
- Total subscription spending overview  
- Category-wise breakdown  
- Visual charts for insights  
- Upcoming renewals tracking  

### 💳 Subscription Management
- Add new subscriptions  
- Edit and update subscriptions  
- Delete subscriptions  
- View detailed subscription information  

### 🔔 Alerts & Notifications
- Renewal reminders  
- Upcoming payment tracking  
- Helps avoid unwanted charges  

### 🎨 UI/UX
- Clean and responsive design  
- Reusable components  
- Smooth user experience  

---

## 🛠️ Tech Stack

**Frontend**
